# XJourney 疆行 · 页面入口地图（按钮可用性对照表）

> 用途：答辩/验收时逐项核对每个界面**从哪里进入**、页内**每个按钮点了去哪**。
> 演示模式（默认开启，个人中心底部可关）下所有按钮均可点：读操作走示例数据，写操作本地模拟成功（提示带"演示模式"字样）；关闭演示模式后全部走真实后端接口。

## 0. 全局顶部导航（所有登录后页面常驻）

| 元素 | 去向 |
|---|---|
| 左上 Logo | `/dashboard` 双圈首页 |
| 「双圈首页」 | `/dashboard`（双圈总入口，斜向分屏） |
| 「学业圈」 | `/dashboard?scene=study`（学业圈信息流） |
| 「生活圈」 | `/dashboard?scene=life`（生活圈信息流） |
| 「通知」/ 铃铛图标 | `/notifications` 通知中心 |
| 顶部搜索框（回车） | `/knowledge?q=关键词` 检索页 |
| 头像 → 个人中心 | `/profile` |
| 头像 → 管理后台 | `/admin/audit`（管理员或演示模式可见） |
| 头像 → 退出登录 | 清登录态 → `/login` |

## 1. 登录 `/login`（首次进入的页面）

| 按钮 | 行为 |
|---|---|
| 中央大 Logo「点击进入」 | 播放 P1→P6 转场，出现登录卡 |
| 「在校生 / 毕业生」切换 | 切换登录身份 tab |
| 「登 录」 | 演示模式：任意账号密码直接进 `/dashboard`；真实模式：调 `/auth/login` |
| 「立即注册」 | `/register` |
| 「忘记密码？」 | 提示联系管理员 |

## 2. 注册 `/register`

| 按钮 | 行为 |
|---|---|
| 「在校生 / 毕业生」 | 切换身份（identityType） |
| 「注 册」 | 演示模式：模拟成功 → `/login`；真实模式：调 `/auth/register` |
| 「返回登录」 | `/login` |

## 3. 双圈首页（总入口）`/dashboard`

| 元素 | 去向 |
|---|---|
| 左半屏「进入生活圈」（整块可点） | `/dashboard?scene=life` |
| 右半屏「进入学业圈」（整块可点） | `/dashboard?scene=study` |
| 底部卡「生活圈动态」条目/更多 | `/dashboard?scene=life` |
| 底部卡「学业圈推荐」条目/更多 | `/dashboard?scene=study` |
| 底部卡快捷入口：经验知识库 | `/knowledge` |
| 　结构化求助 | `/help` |
| 　机会组队 | `/opportunities` |
| 　成长时间线 | `/timeline` |
| 　成长画像 | `/profile` |
| 底部卡通知行 | `/notifications` |

## 4. 圈内信息流 `/dashboard?scene=life|study`

| 元素 | 去向 |
|---|---|
| 横幅「发起求助」 | `/help/create` |
| 横幅「找学习资料」（学业）/「浏览活动机会」（生活） | `/knowledge` / `/opportunities` |
| 左栏快捷入口 ×4（我的求助/经验知识库/机会组队/成长时间线） | `/help` `/knowledge` `/opportunities` `/timeline` |
| 左栏「我的成长进度 → 时间线 ›」 | `/timeline` |
| 信息流卡标题 /「详情」 | 知识条目 → `/knowledge/:id`；求助单 → `/help/:id` |
| 右栏「热门资源/活动 → 更多 ›」及条目 | `/opportunities` |
| 右栏「推荐学习小组/社团 → ＋加入」 | `/opportunities`（组队 tab 所在页） |
| 右栏「圈内通知 → 更多 ›」及条目 | `/notifications` |

## 5. 经验知识库（检索页）`/knowledge`

| 元素 | 行为 |
|---|---|
| 横幅大搜索框「搜索」/回车 | 页内检索（关键词写入搜索历史） |
| 左栏内容类型 ×6 | 按分类过滤并重载列表 |
| 左栏发布时间/排序方式 | 本页筛选态 |
| 结果 tabs（综合/知识条目/求助单） | 切换结果集 |
| 结果卡标题 /「详情」 | `/knowledge/:id` |
| 右栏热门话题（1-5） | 以该话题作为关键词检索 |
| 右栏搜索历史 chip / ×删除 / 清空 | 复搜 / 管理历史 |
| 向下滚动 | 横幅整图自动折叠；回顶展开 |

## 6. 知识详情 `/knowledge/:id`

| 元素 | 行为 |
|---|---|
| 「有用 / 已过时 / 需更新」 | 提交评价（演示模式本地计数+1） |
| 右栏「相关推荐」条目 | `/knowledge/:id`（另一条） |
| 面包屑「学业圈」 | `/dashboard?scene=study` |

## 7. 结构化求助列表 `/help`

| 元素 | 去向 |
|---|---|
| 横幅「发起求助」 | `/help/create` |
| 状态 tabs（全部/进行中/已采纳/已关闭） | 过滤列表 |
| 排序（最新/最久等待/无人回应） | 循环切换排序 |
| 求助卡标题 /「去回答」 | `/help/:id` |

## 8. 发起求助 `/help/create`

| 按钮 | 行为 |
|---|---|
| 「发布求助」 | 演示模式：模拟发布成功 → `/help`；真实模式：POST `/help-tickets` |
| 面包屑「学业圈」 | `/dashboard?scene=study` |

## 9. 求助详情 `/help/:id`（核心闭环页）

| 元素 | 行为 |
|---|---|
| 「提交回答」（三段式表单） | 演示模式：新回答即时上屏；真实模式：POST answers |
| 回答卡「采纳」（提问人视角） | 演示模式：该回答标记已采纳、单据转已采纳；真实模式：PATCH adopt |
| 「追问」 | 演示模式：追问即时上屏（限 3 次提示）；真实模式：POST followups |
| 右栏「相关求助」条目 | `/help/:id` |
| 「返回求助列表」 | `/help` |

## 10. 机会与组队 `/opportunities`

| 元素 | 行为 |
|---|---|
| 「机会列表 / 组队」tabs | 切换视图 |
| 类型 tabs（全部/竞赛/大创/实习/讲座） | 过滤 |
| 机会卡「报名」 | 演示模式：报名人数+1 提示成功；真实模式：PATCH apply |

## 11. 成长时间线 `/timeline`

| 元素 | 行为 |
|---|---|
| 路线切换（考研/就业/竞赛/考公） | 重载该路线节点（演示模式本地切换） |
| 成长曲线上的节点 | 平滑滚动定位到对应节点卡 |
| 左栏「下一步建议 → 去完成」 | 滚动定位到该节点 |
| 节点卡「标记完成」 | 演示模式：本地置已完成并重算进度；真实模式：PATCH progress |
| 节点卡「关联资源」 | 提示（资源清单整理中） |

## 12. 通知中心 `/notifications`

| 元素 | 行为 |
|---|---|
| 左列表条目 | 右侧显示详情（选中高亮） |
| 「全部已读」 | 演示模式本地全部置已读 |
| 详情「查看相关求助/知识」 | 按类型跳 `/help/:id` 或 `/knowledge/:id` |
| 详情「标记已读」 | 演示模式本地置已读 |

## 13. 个人中心 `/profile`

| 元素 | 行为 |
|---|---|
| 横幅「切换生活圈/学业圈」 | 本页绿蓝双主题切换（`?scene=life`） |
| 「编辑资料」 | 提示（开发中） |
| 左栏「成长轨迹 → 查看全部 ›」 | `/timeline` |
| 中栏「创建内容 ›」 | 学业态 → `/knowledge`；生活态 → `/help/create` |
| 右栏「近期动态 → 查看全部 ›」 | `/notifications` |
| **底部「演示模式」开关** | 开：全站示例数据；关：连真实后端 |

## 14. 管理后台 · 审核 `/admin/audit`（头像菜单进入）

| 元素 | 行为 |
|---|---|
| 类型 tabs（知识候选/身份认证等） | 过滤队列 |
| 审核卡「通过 / 退回」 | 演示模式：本地移出队列并更新统计；真实模式：PATCH decide |
